# Agentic AI Course (6-Week Program)

A comprehensive program covering Python fundamentals through production deployment of AI agents.

## Course Materials

| Week | Topic | Topics |
|------|-------| -------|
| 1 | [Python for Agentic AI](docs/w1_python.md) | basics of python, list, function and api |
| 2 | [Machine Learning Essentials](docs/w2_ml.md) | ML basica, regression, logistic models , pickle files |
| 3 | [RAG (Retrieval-Augmented Generation)](docs/w3_rag.md) | private data, RAG implementation, PDF , embeddings |
| 4 | [MCP Server ](docs/w4_mcp.md) | tools, MCP implementation |
| 5 | [Agent Architecture](docs/w5_agents.md) | Agents, memory, tools , multi agents |
| 6 | [Agent Deployment](docs/w6_deploy.md) | dockers , google vertex AI |

## Learning Path

```
Week 1: Python Basics
   ↓
Week 2: Machine Learning
   ↓
Week 3: RAG Systems
   ↓
Week 4: MCP Server & Agent Architecture
   ↓
Week 5: Multi-Agent Systems
   ↓
Week 6: Production Deployment
```

## Prerequisites
